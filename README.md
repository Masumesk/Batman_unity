Batman in Gotham City
  A 2D game where the player can switch between batman and batmobil , move around the scene, control speed ,switch between different game states ( Normal / stealth /Alert) 
  and control batsignal in the sky.

Inputs and Controles 
  A/D -> <- : moving right and left(batmobil and batman)
  W / S ↑  ↓ : moving up and down(batman)
  Shift : Boost
  M : switch between batman and batmobil
  N : normal state
  C : Stealth state
  Space : Alert state
  B : turn the bat signal on and off

Game environment:
  the game is set in gotham city with city sound playing in the background. 

States:
  Each state affects gameplay:
    Normal : normal speed and lighting
    Stealth : slower speed and dim lighting
    Alert : fast speed ,flashing light, Alert sound

Batsignal:
  rotate around it's own axis and move around the sky
